## git how to
# Without proxy
1. git init
2. git config --global --add http.proxy ""
3. git config --global --add https.proxy ""
4. git clone https://github.com/chinthakaak/test_projects.git
5. git add testfiletoadd.txt
6. git commit testfiletoadd.txt -m "test file"
7. git push
